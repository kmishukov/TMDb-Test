Главный экран: ViewController + TableView = Top Rated Movies ;
LeftBarButtonItem - обновить список;
RightBarButtonItem - показать список с избранными фильмами;
TableView строится по массиву moviesArray типа [Movie], подгрузка данных осуществляется с пейджингом по мере прокрутки списка;

Кастомный поиск (пришлось сделать из-за выбранного мною дизайна - oh god why) по базе данных movieDb.org; При нажатии на поисковик, tableView начинает строиться по массиву searchResultsArray типа [Movie]. Запрос к серверу (func updateSearchResults) осуществляется по мере набора символов в строку поиска (customSearchBar) с использованием функции NSObject.cancelPreviousPerformRequests, подгрузка данных осуществляется с пейджингом по мере прокрутки списка;

При нажатии на ячейку фильма в любой из вышеперечисленных таблиц - переходим на страницу детальной информации о фильме. Сюда посылается объект типа Movie и на его основе строится интерфейс; Посылается запрос на постер с большей шириной;

Избранное (Realm)
Список экрана избранное строиться на отсортированном массиве объектов MovieObject из базы Realm. Объект может быть добавлен из таблиц topRated / searchResults (через звездочку) либо со страницы с детальной информацией о фильме. При отключенном интернете можно просматривать список избранного и страницы с детальной информации о каждом из списка избранного фильме. Постеры фильмов в Realm не сохраняются, только их urlString.
